README

Project: GameOfSticks

Description:
	A game played between the user and the computer. Players take turns
		picking up 1-3 sticks from the original pile. The player who
		picks up the last stick loses the game.
	
Files:
	sticks.c
	
Execution:
	Compile using gcc sticks.c and then a.out to run.
	The game will ask for an initial pile size between 10 and 100.
	Then the game will repeatedly ask the user to pick up 1-3 sticks,
		alternating with computer picks, until the pile is empty.
